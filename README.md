# MERN Hotel Booking App

## 📌 Tech Stack
- MongoDB
- Express
- React (Vite)
- Node.js

## ⚙️ Installation

### Clone the repo
git clone <your-repo-link>

### Client Setup
cd client
npm install
npm run dev

### Server Setup
cd server
npm install
npm start

## 📂 Project Structure
client → Frontend (React)
server → Backend (Node + Express)
